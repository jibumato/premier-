# プルミエ！ — コスプレイヤー交流マッチング (プロトタイプ実装)

コスプレイヤー同士（カメラマンも参加可）が好きな作品でつながり、併せ撮影の
企画・募集・応募ができる交流プラットフォーム「プルミエ！」のモバイル画面実装です。

`design_handoff_premier_prototype`（Anthropic 社内デザインツールの `.dc.html`
リファレンス）を **見た目・レイアウト・状態遷移の仕様書** として参照し、本番用の
コードとして新規に実装し直したものです。デザインファイルはそのまま流用していません。

## 技術スタック

- **Next.js 15**（App Router）+ **React 19** + **TypeScript**
- モバイルWeb（390px 幅、デザインが対応する唯一のブレークポイント）
- Web フォントは Google Fonts の `Zen Kaku Gothic New` / `Zen Maru Gothic` を
  ランタイム `<link>` で読み込み（ビルド時の外部依存なし）
- アイコンはすべてインライン SVG（Feather 風ライン、stroke-width 1.6–2）

> 未確定だったフロントエンド技術スタックは、ハンドオフ README の推奨
> （「未確定ならモバイルWeb兼用の React（Next.js等）」）に従い Next.js を採用。
> 状態機械・データ・トークンは技術非依存に切り出してあり、React Native /
> Flutter への移植も容易です。

## 実装済み画面（8 スクリーン）

| screen key | 画面 | 概要 |
| --- | --- | --- |
| `home` | ホーム | 検索入口・人気作品チップ・併せ募集フィード・投稿グリッド |
| `search` | さがす | 地域フィルタ（チップ）／3 タブ／結果リスト／空状態カード |
| `detail` | 併せ募集 詳細＋応募 | ヒーロー・主催者カード・基本情報グリッド・募集キャラ・応募ボタン |
| `applied` | 応募完了 | チェックマーク確認画面 |
| `create` | 併せ募集 作成 | プログレス・参考画像・入力欄・世界観タグ・トグル 2 種 |
| `created` | 公開完了 | 公開確認画面 |
| `notify` | おしらせ | 通知リスト（未読ハイライト） |
| `profile` | マイページ | カバー／グラデーションリングアバター／称号／スタッツ／応援リンク／応援ギフト／ギャラリー |

## 状態遷移・振る舞い

`clickable_prototype.dc.html` の DCLogic 状態機械を移植しています。

- `screen` — 現在画面、`tab` — 下部ナビのアクティブタブ、`history` — 戻る用スタック、
  `region` — 検索の地域フィルタ（初期値「すべて」）。
- 下部タブナビ（ホーム / さがす / ＋作成 / おしらせ / マイページ）。中央 ＋ は
  常時ラベンダーグラデーションの FAB。
- 地域チップのタップで結果リストと件数表示がクライアント側で再計算され、
  0 件時は空状態カードを表示。
- 作成フォームのトグル（女性限定 / 初心者歓迎）はインタラクティブ。

## 運営方針の反映（Constraints）

ワンオペ運営前提のため、ハンドオフの制約を UI に反映しています。

- 応援ギフト（投げ銭）UI は **将来機能のプレースホルダ**（サイト内決済は未実装）。
- 実装対象の収益化導線は **外部の応援リンク（Fantia / pixivFANBOX / Skeb）のみ**。
- 応援リンクは **年齢確認済み（18歳以上）ユーザーにのみ表示** するゾーニングを実装。
  「サイト内にアダルトコンテンツはありません」を明記。

## ディレクトリ構成

```
src/
  app/
    layout.tsx        # フォント読み込み・メタデータ
    globals.css       # リセット＋キャンバス背景
    page.tsx          # AppShell を描画
  lib/
    tokens.ts         # デザイントークン（色・角丸・影・余白）
    types.ts          # 画面キー・データ型
    data.ts           # モックデータ（本番は API 由来）
  components/
    AppRouter.tsx     # クライアント側ルーター（screen/tab/history/region）
    AppShell.tsx      # 画面スイッチャ
    PhoneFrame.tsx    # 390px フレーム＋ステータスバー＋下部ナビ
    StatusBar.tsx
    BottomNav.tsx
    ImageSlot.tsx     # 画像プレースホルダ（本番は Image + オブジェクトストレージ）
    icons.tsx         # インライン SVG アイコン
    ui.tsx            # 共通 UI（見出し・プライマリボタン）
    screens/          # 8 画面
```

## 開発

```bash
npm install
npm run dev     # http://localhost:3000
npm run build   # 本番ビルド
npm run lint
```
