## 現状維持でよいと確認された点

| # | 論点 | 弁護士の要点 |
| --- | --- | --- |
| ① | 越境移転（Supabase） | DBが国内で処理・保管され米国法人が取得・利用しないなら**通常は該当しない**。ただし米国側から実質的にアクセスできる運用なら該当し得る（→本サービスは保守運用でのアクセス可能性を開示済みで、保守的に越境移転として記載を維持） |
| ③ | 本人確認書類（削除） | 確認後削除・フラグのみ保持は**データ最小化の観点からも適切** |
| ④ | 本人確認書類（マイナンバー） | 取得しない運用は**妥当** |
| ⑤ | 本人確認（目視） | 18歳以上確認目的なら**目視でも一定の合理性**（eKYCは信頼性が高いが必須ではない） |
| ⑦ | 撮影と掲載の同意 | 別の同意として扱うのが**望ましい**（現行どおり） |
| ⑧ | 権利の併存 | カメラマンの著作権と被写体の肖像権の併存整理で**問題なし** |
| ⑨ | 版権配慮 | 「ユーザーが権利侵害しない」「運営は削除・利用停止できる」を定めるのが望ましい（規約に既定・維持） |
| ⑬ | 所在地開示 | 無償のみなら**必須ではない**（有料開始時に特商法表示が必要） |
| ⑮ | 個人情報保護管理者 | 個人事業者に**設置義務なし**。問い合わせ窓口の明記で足りる（対応済み） |

## 残る運用上の留意点

- **Supabase の米国からのアクセス**：越境移転の該当性は「実質的なアクセスの有無」で変わるため、
  保守的に越境移転として開示を維持している（①）。
- 本メモは要約。公開直前に、最終版の規約・プライバシーを弁護士に最終確認してもらうのが望ましい。
